## 2D Minecraft
A flat version of the most popular game in the world!
### Controls
W, A, S, D to move\
F3 to show debug\
Click to place/break blocks\
1-8 to select blocks
### Recent changes
1. Texture update\
&nbsp;&nbsp;&nbsp;&nbsp;Changed block textures to look more like in the real game.
2. Building update\
&nbsp;&nbsp;&nbsp;&nbsp;Click on a block to break it.\
&nbsp;&nbsp;&nbsp;&nbsp;Use number keys (1-8) to select building blocks, click on an air block to place.
3. FX Update\
&nbsp;&nbsp;&nbsp;&nbsp;Breaking a block will now give off particles.\
&nbsp;&nbsp;&nbsp;&nbsp;You will now see the type of block you are going to place
4. New blocks!\
&nbsp;&nbsp;&nbsp;&nbsp;Added sand, gravel and water.\
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;**Sand:**\
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Spawns naturally at a specific Y level.\
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Currently serves as another decoration block.\
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Has no gravity at the moment.\
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;**Gravel:**\
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Currently serves as another decoration block.\
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Has no gravity at the moment.\
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Does not naturally appear yet.\
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;**Water:**\
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Spawns naturally at a specific Y level.\
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Spreads into all nearby air blocks every 0.2 sec.
5. Added an icon!\
&nbsp;&nbsp;&nbsp;&nbsp;The program now has a grass block as its window icon.
